1. Nền tảng .NET Framework là gì? 
Là môi trường đa ngôn ngữ để xây dựng, triển khai và chạy các ứng dụng do hãng Microsoft phát
triển.
.NET Framework cho phép phát triển các ứng dụng chạy trên nhiều môi trường khác nhau như thiết bị
di động, máy tính để bàn, điện thoại thông minh, dịch vụ web…

2. Các công nghệ cốt lõi sử dụng trong .NET Framework là gì?
Là ngôn ngữ lập trình sẽ được dịch sang ngôn ngữ Microsoft Intermediate Language(MSIL)
Sau đó nhờ Common Language Runtime dịch sang mã máy

o Các ngôn ngữ lập trình
o Thư viện các lớp(Framework Class Library - FCL)
o The Common Language Runtime(CLR)
